# chatwork-skin-slack

Chrome extension to apply Slack skin to Chatwork

## Getting started

```
git clone git@github.com:Leko/chatwork-skin-slack.git
cd chatwork-skin-slack
npm i
gulp
```

Then open `chrome://extensions/` and import `dist` directory.
